# GlamAI

## AI-Powered Skincare Intelligence Platform

GlamAI is a full-stack AI-powered skincare platform that combines Computer Vision, Deep Learning, Conversational AI, and E-Commerce to provide personalized skincare guidance and product recommendations.

## Features

- AI-Based Skin Type Detection
- TensorFlow + MobileNetV2 Deep Learning Model
- OpenCV Image Processing Pipeline
- Sensitive Skin Detection using Redness Analysis
- Personalized Product Recommendations
- Intelligent Skincare Chatbot
- Google Authentication
- JWT Secure Authentication
- Shopping Cart Management
- Razorpay Payment Gateway
- Order Tracking System
- Admin Dashboard
- Blog & Educational Content System
